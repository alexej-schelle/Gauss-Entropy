# This folder contains a summary of the most important source files for the QMCM Software Application.

- bec_symmetry_breaking.py : 

  Refers to a routine which calculates the quantum field of a Bose-Einstein condensate 
  with respect to the broken gauge symmetry.  
  
- bec_time_uncertainty.py : 

  Refers to a routine which calculates the coherence time distribution of the BEC quantum field.  

- superimposed_bec.py : 

  Refers to a routine which calculates the quantum field of two superimposed BECs.
